2. Purpose of the Application

The application allows users to register the purchase date of gas cylinders, start and end usage dates, purchase price, approximate purchase location, and view usage statistics and duration history. The app may also send local reminder notifications.

3. Data Collected

The application stores exclusively on the user's device:

  • Purchase date
  • Start and end usage dates
  • Manually entered price
  • Manually selected location
  • Gas cylinder history records

4. Device Permissions

Location: Used solely to display nearby gas stations and assist manual selection.

Notifications: Used only for local reminders regarding estimated replacement dates.

5. Data Storage

All data is stored locally on the user's device using UserDefaults and internal iOS storage. No external servers are used and no data is shared with third parties.

6. Data Sharing

The application does not sell, share, or transfer personal data to third parties.

7. Security

All information remains protected within the user's device under the security measures provided by the iOS operating system.

8. User Rights

Users may delete all stored data by uninstalling the application or revoke permissions through device settings.
